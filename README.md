Swift
=========

An Ansible role for installing Apple Swift.

Requirements
------------

Ubuntu 12.04 or higher

Role Variables
--------------

- `version`
  - default: `2.2-SNAPSHOT-2016-01-06-a`
- `platform`
  - default: `ubuntu15.10`
- `platform_short`
  - default: `ubuntu1510`
- root
  - default: `/opt/swift`
